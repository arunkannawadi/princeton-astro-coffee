The Astro-Coffee Server is a small platform to view and vote on arXiv astro-ph papers, and helps to organize paper discussions. It's a bit like VoxCharta, but much simpler, and can be fully installed on a local department server.

Astro-Coffee administrator

Code and bug-reports

The server's code is hosted on Github, and it's meant to be simple to install and use. Please let Waqas Bhatti know if something doesn't look right, or is completely broken, or if you have questions about the code.
