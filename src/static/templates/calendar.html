This page shows the calendar of astro-coffee hosts. If you encounter a scheduling conflict that makes you unable to host one of your scheduled sessions please arrange to swap dates with another host, and reach out to the coffee adminstrator, Bruce Draine to implement the change. You can import events from the teamup calendar into your personal calendars by clicking a specific event and use the "Share" dropdown menu to select which application to export the event to.

A pdf version of the calendar can be found here: astrocoffee_2021.pdf or by using the burger menu of the teamup calendar.
